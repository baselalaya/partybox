# **App Name**: Party Box

## Core Features:

- Booth Listing: Displays a list of available photo booths with key details.
- Event Catalog: Shows available Events to which photo booths can be added
- Dynamic Page Generation: Generates static pages for booths, events, and locations from WordPress data.
- Contact Form Submission: Enables users to submit contact information via a form.
- SEO Optimization: Generates SEO metadata and structured data for all pages using data pulled from the CMS. Dynamically optimizes keywords based on content using a tool.  This uses 'next-sitemap' or a custom solution to autogenerate 'sitemap.xml', and 'robots.txt'.

## Style Guidelines:

- Primary color: Vibrant pink (#E91E63) to convey the fun and energy of photo booth experiences.
- Background color: Dark gray (#212121) to provide a modern, premium feel and make the accent color pop.
- Accent color: Purple (#9C27B0) for secondary highlights and calls to action, creating visual interest and contrast.
- Headline font: 'Poppins', a geometric sans-serif, providing a modern, precise, fashionable look.
- Body font: 'Inter', a grotesque-style sans-serif with a modern look. Pairs well with Poppins if needed.
- Use clean, modern icons relevant to photo booths, events, and locations. Icons should be consistent in style and easily recognizable.
- Mobile-first, fully responsive layout with clear visual hierarchy. Use plenty of whitespace for readability and a premium feel.
- Subtle hover animations and transitions to enhance user experience without impacting performance. Use CSS transitions and lightweight animation libraries.
